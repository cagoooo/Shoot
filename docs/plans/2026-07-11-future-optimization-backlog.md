# 地球守護隊：後續優化與功能待辦

更新日期：2026-07-15  
適用版本：`0.1.0-vertical-slice` 之後的九大世界版

這份清單將「首版能不能安心進教室」與「讓遊戲更豐富」分開。P0 是進入校內小規模試玩前優先完成的項目；P1 是驗收穩定後，最值得投資的內容擴充。所有項目維持國小中高年級可理解、非血腥、以 SDGs 與自然科學／工程設計為主軸。

## 已完成的基礎

- 〈垃圾風暴救援行動〉可完成基地、工具組裝、兩條探索路線、分類挑戰、能源選擇、淨化與撤離。
- 已串聯 SDG 7、12、13；具備任務回顧、徽章、永續行動卡與本機進度。
- GitHub Pages、手機與平板瀏覽器基線、PWA 更新提示、favicon 與社群分享卡已完成。
- **九大 SDGs 世界全部可玩（9／9）**：含地球行動地圖、解鎖進度、各世界故事與三段科學任務；第 2 關水滴守護、第 3 關綠能社區有專屬 3D 場景與玩法。
- **課堂 P0 第一輪**：真機驗收表、教師引導單與學生回饋卡已備妥；觸控設定（瞄準速度、左右手、一鍵重設）與圖卡式任務引導初版完成。
- **體驗打磨（2026-07-12）**：3D 目標地標連動任務操作、過濾順序引導回饋、多選任務可見回饋、手機觸控滑動轉視角、響應式基地導覽。
- 正式 BGM 7 首（Suno Pro Plan）已部署並有授權台帳與 SHA-256 驗證；遊戲內可切換音樂、任務報告有專屬配樂。
- 自動化檢查目前包含 136 個單元測試（61 個測試檔）與 18 個瀏覽器端對端測試情境。

## 建議的選擇順序（2026-07-15 更新）

1. P0-1 與 P0-4 只剩「實際拿裝置、帶學生玩」這一步，建議最先安排；程式端可先補 P0-5 的音量記憶與色彩輔助、P0-6 的發布檢查表。
2. 內容方向建議從 P1-3 九大世界深化開始（第 4 到第 9 關逐關補專屬場景與玩法），或先做 P1-5 學習報告升級讓試玩資料更有用。
3. 再依教學使用情境選 P1-8 的教師課堂模式或 P1-10 的美術升級。
4. 多人連線只在單人內容、真機效能與課堂流程穩定後，再開始 P1-12 的預研。

## 本輪暫不建議優先投入

- 公開多人連線、排行榜、帳號系統與聊天功能：會增加兒少隱私、管理與維運風險。
- 一次完成 17 個 SDGs 世界：應以每個世界都有清楚玩法與可驗證學習目標為原則。
- 寫實槍械、血腥畫面或高壓驚嚇機制：不符合本專案國小教育與安全定位。
