# 地球守護隊：後續優化與功能待辦

更新日期：2026-07-11  
適用版本：`0.1.0-vertical-slice`

這份清單將「首版能不能安心進教室」與「讓遊戲更豐富」分開。P0 是進入校內小規模試玩前優先完成的項目；P1 是驗收穩定後，最值得投資的內容擴充。所有項目維持國小中高年級可理解、非血腥、以 SDGs 與自然科學／工程設計為主軸。

## 已完成的基礎

- 〈垃圾風暴救援行動〉可完成基地、工具組裝、兩條探索路線、分類挑戰、能源選擇、淨化與撤離。
- 已串聯 SDG 7、12、13；具備任務回顧、徽章、永續行動卡與本機進度。
- GitHub Pages、手機與平板瀏覽器基線、PWA 更新提示、favicon 與社群分享卡已完成。
- 自動化檢查目前包含 125 個單元測試與 17 個瀏覽器端對端測試。

## 建議的選擇順序

1. 先選 P0-1、P0-2、P0-3、P0-4，讓首個任務成為可安全試玩的課堂版本。
2. 接著選一個內容方向：P1-1 的水資源任務或 P1-2 的綠能社區任務。
3. 再依教學使用情境選 P1-5 的學習報告或 P1-8 的教師課堂模式。
4. 多人連線只在單人內容、真機效能與課堂流程穩定後，再開始 P1-12 的預研。

## 本輪暫不建議優先投入

- 公開多人連線、排行榜、帳號系統與聊天功能：會增加兒少隱私、管理與維運風險。
- 一次完成 17 個 SDGs 世界：應以每個世界都有清楚玩法與可驗證學習目標為原則。
- 寫實槍械、血腥畫面或高壓驚嚇機制：不符合本專案國小教育與安全定位。
