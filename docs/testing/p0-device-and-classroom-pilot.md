# P0 真機驗收與課堂小組試玩

版本：`0.1.0-vertical-slice`  
適用對象：桃園市龍潭區石門國民小學的國小中、高年級小組試玩

這份表單將瀏覽器自動測試與真機驗收分開。只有實際載具完成紀錄，才可將該裝置標示為「已驗收」。

### 每台載具必測項目

- [ ] 開啟 https://cagoooo.github.io/Shoot/ 後，標題、按鈕與圖片沒有被裁切。
- [ ] 選擇「中年級輔助」後，學生能看懂「現在要做什麼」圖卡。
- [ ] 觸控搖桿、能量工具、互動按鈕都能按到；左右手模式與瞄準速度可調整、可重設。
- [ ] 從基地完成一條任務路線，包含分類、能源選擇與撤離。
- [ ] 切換到其他 App 或鎖定螢幕後回來，沒有遺失進度或卡住。
- [ ] BGM 可以開關，關閉後仍可完成任務。
- [ ] 安裝為 PWA 後，發布新版時能看到「新版本已準備好」提示，按「立即更新」可正常回到遊戲。
- [ ] 連續遊玩 15 分鐘沒有嚴重發熱、明顯掉幀或當機。

## B. 15–20 分鐘小組試玩流程

| 時間 | 教師引導 | 學生任務 | 觀察重點 |
| --- | --- | --- | --- |
| 0–3 分鐘 | 問：「回收物放錯地方會怎樣？」介紹 SDG 12。 | 選擇中年級輔助或高年級標準模式。 | 是否能自行開始。 |
| 3–6 分鐘 | 示範圖卡、搖桿與能量工具各一次。 | 在基地與試玩區練習。 | 按鍵是否太小、方向是否難控制。 |
| 6–15 分鐘 | 不直接給答案，改問「提示告訴你什麼？」 | 完成分類、能源選擇與撤離。 | 能否讀懂提示、是否願意比較選擇。 |
| 15–20 分鐘 | 引導回顧：「哪一個選擇最省能源？為什麼？」 | 完成回饋卡。 | 能否把遊戲選擇連到生活行動。 |

## D. 教師快速判讀

- 若多數學生在 3 分鐘內無法開始：優先調整 P0-3 圖卡與開始畫面。
- 若多數學生找不到按鍵或持續誤觸：優先調整 P0-2 觸控位置與尺寸。
- 若學生只能答對、說不出理由：強化任務後的「小小科學發現」與討論題。
- 若某一載具掉幀、當機或無法續玩：標示為 P0 阻斷問題，先修復再擴大試玩。
