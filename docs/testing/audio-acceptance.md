# 遊戲音訊驗收清單

## 自動規則

- [ ] 使用者按下開始前，不建立 AudioContext 或自動播放。
- [ ] 語音／旁白播放時，BGM 增益降至原本的 35%。
- [ ] 語音結束後，BGM 恢復原音量。
- [ ] 分頁隱藏時暫停，返回頁面後恢復。
- [ ] 靜音時 BGM 與音效都為零。
- [ ] 場景切換使用 900 ms 交叉淡化。
- [ ] `approved` 資產至少有一個實際來源檔。

## 桌機、平板與手機人工驗收

- [ ] Chrome／Edge／Safari 第一次進站保持安靜，按開始後才可播放。
- [ ] iPad／iPhone 靜音鍵、音量鍵與藍牙耳機行為合理。
- [ ] Android Chrome 切換分頁後不會重疊播放兩首 BGM。
- [ ] 觸控操作時不會因連續點擊重複建立 AudioContext。
- [ ] 基地、探索、黃色警示、頭目、撤離、成功、報告曲風可辨識。
- [ ] BGM 不蓋過任務文字、教師口頭指令或遊戲旁白。
- [ ] 沒有突發尖銳聲、恐怖聲、槍響擬真聲或血腥音效。
- [ ] 循環接點沒有爆音、空白斷點或明顯音量跳動。
- [ ] 音樂關閉後，重要警示仍有視覺與文字提示。

## Suno 正式檔案上架閘門

- [ ] 由阿凱老師的成人帳號取得，不涉及學生帳號。
- [ ] 作品 ID、原始連結、日期、付費方案、提示詞與模型已填。
- [ ] 原始檔與遊戲檔 SHA-256 已填。
- [ ] 剪輯與循環處理有紀錄。
- [ ] 保存產生當日的條款／授權說明快照。
- [ ] 審核人確認適合國小中高年級與校園播放。
- [ ] manifest 狀態由 `awaiting-audited-file` 改成 `approved`。
- [ ] GitHub Pages 實際部署後重新測試來源路徑與 CORS。
