# Suno BGM 生成提示詞：地球守護隊：能量大作戰

適用對象：阿凱老師的成人 Suno 帳號。學生不登入、不操作 Suno，也不需要看到這份製作文件。

## 共通設定

- 使用當下具有遊戲商用權利的成人付費方案；生成當日先截圖保存方案和條款頁面。
- 啟用 **Instrumental／純音樂**，不要填歌詞。
- 每首先產生 2–4 個版本，再依「可閱讀、不恐怖、不搶戲」挑選。
- 不要填寫現有歌手、樂團、作品或系列名稱；每首都要求 original composition。
- 建議輸出 WAV 或最高品質音檔，後製成 OGG 與 MP3；音量以約 -16 LUFS 為起點，避免壓過閱讀與任務提示。
- 每次生成後立刻在 `suno-asset-ledger.md` 填入作品連結、日期、方案、模型、原始檔 SHA-256 與條款快照位置。

## 生成後的快速篩選

每首先播放 30 秒，逐項確認：

1. 沒有人聲、口白或可辨識歌詞。
2. 沒有突發巨響、尖銳警報、恐怖音色或強烈戰鬥感。
3. 以低音量播放時仍不干擾學生閱讀。
4. 循環點不會明顯斷裂；成功短曲有明確收尾。
5. 可對應該場景的「基地、探索、觀察、修復、撤離、肯定、反思」情緒。

通過後才進行剪輯、轉檔、雜湊、台帳登錄與遊戲整合。
