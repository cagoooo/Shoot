## 驗收原則

- 主要按鈕不得超出畫面，觸控區至少約 44×44 像素。
- 3D 畫面載入失敗或教材檔暫時不可用時，學生能回基地，不會卡死。
- 低於 28 FPS 持續約 5 秒才降低一級畫質；同一次低效能事件只降一次。
- 實機試玩需補記載具型號、作業系統、瀏覽器版本、平均 FPS、聲音與觸控結果。

> 自動化裝置模擬不能取代實機測試；上表刻意分開標示，避免把模擬結果誤寫成實機通過。
